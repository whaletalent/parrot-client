# Parrot: Simple chat application using VueJs, Vuetify, NodeJs, and SocketIo

## Project install:
```
Open terminal in parrot current folder and run:
npm install
```

### Post install Run:
```
npm run server
```

### Compiles and minifies for production:
```
npm run build
```

### Online Heroku Demo:

https://parrot-chat-server.herokuapp.com
